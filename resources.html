FAL provides various resources for performing archaeological research. These resources are made feely available for use and distribution, and make use of freely available, open-source tools. All resources are available as-is. We do not provide technical or trouble-shooting support.

Code

  • Radiocarbon calibration
    Used to quickly return calibrated radiocarbon dates with violin plots using Bchron calibration package as implemented in R statistical computing language.

  • Photo Point Picker
    Used to derive ground-control point image coordinates from photographs and pair them with real-world coordinates for use in OpenDroneMap. R statistical computing language.
  • JabRef style files (see Bibliography management tutorial on this page)
    • American Anthropologist
    • Society for American Archaeology journals
    • Journal of Archaeological Method and Theory
    • Journal of Archaeological Research
    • Journal of Archaeological Science journals
    • Nature journals
    • PLoS journals
    • Royal Society journals
    • Science and Proceedings of the National Academy of Sciences journals
  • Projectile target code for experimental projectile (archery, atlatl, spear, etc.) research. Code may be adapted for your own purposes, or the image below can be printed as is (be sure to print at 100% scale). Please cite Haas 2024, PaleoAmerica.

  • USGS quad map decollar tool used to remove collar from USGS quad maps so that adjacent quads can be seamlessly stiched. Requires geotif format and associated xml file, which can be downloaded at the USGS topoView website. The code is implemented in R statistical computing language, and requires the Terra and XML packages.
