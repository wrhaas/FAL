FAL provides various resources for performing archaeological research. These resources are made feely available for use and distribution, and make use of freely available, open-source tools. All resources are available as-is. We do not provide technical or trouble-shooting support.

Tutorials

  • Drafting an article manuscript using LibreOffice Writer
  • Bibliography management with JabRef
  • Spreadsheets with LibreOffice Calc
  • Basic statistical programming with R
  • Basic image editing with Gimp

Code

  • Radiocarbon calibration
    Used to quickly return calibrated radiocarbon dates and plots using Bchron calibration package as implemented in R statistical computing language.
  • Radiocarbon violin plot
    Used to create a violin plot of calibrated radiocarbon dates using Bchron calibration package as implemented in R statistical computing language.
  • Photo Point Picker
    Used to derive ground-control point image coordinates from photographs and pair them with real-world coordinates for use in OpenDroneMap. R statistical computing language.
